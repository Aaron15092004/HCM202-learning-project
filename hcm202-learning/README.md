# HCM202 Learning

Ung dung hoc tap React/Vite cho mon **Tu tuong Ho Chi Minh**.

## Noi dung chinh

- 7 chu de kien thuc HCM202.
- Luyen tap trac nghiem theo chu de.
- The ghi nho bang hinh anh chu de Ho Chi Minh.
- Giao dien do vang, dong bo voi nhan dien mon hoc.

## Lenh phat trien

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Cau truc du an

- `src/data/`: du lieu bai hoc, cau hoi va anh flip-card.
- `src/pages/`: cac trang chinh cua ung dung.
- `src/components/home/`: giao dien trang chu.
- `src/styles/`: theme va CSS theo tung khu vuc.
